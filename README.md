# вҡҷпёҸ Installation & Setup

## 1пёҸвғЈ Clone Repository


---

## 2пёҸвғЈ Setup Backend

```bash
cd backend
npm install
node server.js
```

Backend will run on:

```bash
http://localhost:5000
```

---

## 3пёҸвғЈ Setup Frontend

Open a new terminal:


```bash

```
